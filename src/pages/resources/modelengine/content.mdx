# 欢迎来到 ModelEngine ！

最全面、灵活和先进的实体模型引擎

# 简介

你是否觉得 Minecraft 有点单调？是否想拥有一个更多实体模型的体验，
但又不想让玩家下载 MOD 模组？你是否只想自己自定义模型？如果是的话，那么 ModelEngine 就是为你而生！

ModelEngine 是一个插件，让你可以添加具有完整动画支持的自定义实体模型。
它为你处理了所有困难的部分，并弥合了你和你梦寐以求的服务器之间的差距。
它是第一个也是唯一一个无需模组即可创建与 MOD 模组无异的生物的实体模型引擎。
